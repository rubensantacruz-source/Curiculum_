# Currículum Web de Ruben Santa Cruz

Este proyecto contiene un currículum vitae en formato web, diseñado para mostrar tu perfil profesional como diseñador y desarrollador web.

## Archivos principales

- `curriculum.html` - Página HTML principal del currículum.
- `curriculum.css` - Estilos en blanco y negro con diseño responsivo y detalles sutiles.
- `imagenes/` - Carpeta de recursos de imagen, incluyendo la foto de perfil / fondo utilizado en el encabezado.

## Características

- Diseño elegante y formal en blanco y negro.
- Perfil con foto y datos de contacto.
- Secciones de "Sobre mí", "Habilidades", "Experiencia", "Proyectos destacados" y "Educación".
- Responsive: adaptable a móviles, tablets y escritorio.
- Botón "Disponible para trabajos" que abre WhatsApp para contacto directo.
- Favicon personalizado usando la imagen del perfil.

## Cómo usar

1. Abre `curriculum.html` en tu navegador.
2. Reemplaza el texto de ejemplo por tu información real.
3. Si deseas usar tu propia foto, coloca el archivo en la carpeta `imagenes/` y actualiza la ruta en `curriculum.html`.

## Personalización

- Cambia el nombre y datos de contacto en el encabezado.
- Ajusta las secciones de experiencia y educación según tu historial.
- Puedes actualizar el enlace de WhatsApp en `curriculum.html` si cambias de número.

## Notas

- El archivo `README.md` sirve para documentar cómo funciona el proyecto y cómo editar el currículum.
- Puedes usar este proyecto como base para mostrar tu perfil profesional en un portafolio.
